# ☕︎ Organização e Estruturas de Dados

Repositório contendo exercícios da disciplina **Organização e Estruturas de Dados** do segundo ano de **Ciência da Computação** na UNESPAR.

## ☕︎ Sobre o Repositório

Este repositório reúne implementações de conceitos fundamentais de **Organização e Estruturas de Dados**, abordando tópicos como:

- *Revisão da Linguagem C*
- *Algoritmos de Ordenação*
- *Fila*
- *Lista Dinâmica Circular*
- *Lista Dinâmica Sem Sentinela*
- *Lista Duplamente Encadeada*
- *Pilha*
- *Árvore de Busca Binária*
- *Fila de Prioridade*
- *Trabalho Sobre Fila*
- *Pesquisa Binária*
- *Tabela Hash*

# ☕︎ Objetivo

As atividades servem como material de estudo e prática para reforçar os conceitos abordados na disciplina.

## ☕︎ Tecnologia e Linguagens Utilizadas

- C;
- Code::Blocks.
  
## ☕︎ Como Usar

## Como clonar apenas esta pasta

Este repositório contém todas as disciplinas do 2° ano. Se quiser baixar **somente** a pasta "Organização e Estruturas de Dados", use o sparse-checkout do Git:

```bash
git clone --no-checkout https://github.com/MariaCampanaP/Segundo-Ano-de-CCOMP.git
cd Segundo-Ano-de-CCOMP
git sparse-checkout init --cone
git sparse-checkout set "Organização e Estruturas de Dados"
git checkout main
```
