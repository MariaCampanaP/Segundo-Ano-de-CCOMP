1. Crie um novo arquivo chamado ex1.txt e responda a seguinte questão: Dado o código abaixo, diga qual é a complexidade e explique o porquê.

for(int i = 1; i < n; i *= 2){
 for(int j = 0; j < n; j++){
	//operação constante
  }
}

Resposta: A complexidade do código é O(n log n).
O laço externo executa log n vezes, pois i é multiplicado por 2 a cada passo. O laço interno executa n vezes para cada iteração do externo.
Multiplicando os dois: n x log n = O(n log n).
