# ☕︎ Linguagens Formais, Autômatos e Computabilidade

Repositório contendo exercícios da disciplina **Linguagens Formais, Autômatos e Computabilidade** do segundo ano de **Ciência da Computação** na UNESPAR.

## ☕︎ Sobre o Repositório

Este repositório reúne implementações de conceitos fundamentais de **Linguagens Formais, Autômatos e Computabilidade**, abordando tópicos como:

- Expressão Regular em Java.

# ☕︎ Objetivo

As atividades servem como material de estudo e prática para reforçar os conceitos abordados na disciplina.

## ☕︎ Tecnologia e Linguagens Utilizadas

- *Java;*
- *NetBeans.*

## Como clonar apenas esta pasta

Este repositório contém todas as disciplinas do 2° ano. Se quiser baixar **somente** a pasta "Linguagens Formais, Autômatos e Computabilidade", use o sparse-checkout do Git:

```bash
git clone --no-checkout https://github.com/MariaCampanaP/Segundo-Ano-de-CCOMP.git
cd Segundo-Ano-de-CCOMP
git sparse-checkout init --cone
git sparse-checkout set "Linguagens Formais, Autômatos e Computabilidade"
git checkout main
```
