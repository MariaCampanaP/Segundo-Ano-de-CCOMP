# ☕︎ Segundo Ano de CCOMP

Repositório contendo materiais, exercícios e projetos das disciplinas do **segundo ano** do curso de **Ciência da Computação** na UNESPAR.

## ☕︎ Sobre o Repositório

Este repositório reúne o conteúdo estudado ao longo do segundo ano, organizado por disciplina:

- *Teoria dos Grafos*
- *Organização e Estruturas de Dados*
- *Linguagens Formais, Autômatos e Computabilidade*
- *Linguagem Orientada à Objetos*
- *Física*

## ☕︎ Objetivo

Centralizar os códigos, exercícios e anotações produzidos durante o segundo ano, servindo como material de estudo, revisão e portfólio acadêmico.

## ☕︎ Tecnologias Utilizadas

- Java;
- Python;
- C;
- HTML5;
- CSS3;
- JavaScript
- Code::Blocks;
- Visual Studio Code;
- Netbeans;
- Draw.io.

## ☕︎ Como Usar

## Como clonar este repositório

```bash
git clone https://github.com/MariaCampanaP/Segundo-Ano-de-CCOMP.git
```

Isso vai baixar todas as disciplinas do 3° ano.
Se quiser baixar apenas a pasta de uma disciplina específica, veja as instruções de sparse-checkout no README de cada pasta.
