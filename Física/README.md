# ☕︎ Física

Repositório contendo exercícios da disciplina **Física** do segundo ano de **Ciência da Computação** na UNESPAR.

## ☕︎ Sobre o Repositório

Este repositório reúne implementações e simulações de conceitos fundamentais da **Física**, abordando tópicos como:

- *Cinemática*
- *Movimento Retilíneo Uniforme (MRU) e Uniformemente Variado (MRUV)*
- *Leis de Newton*
- *Movimento de Projéteis*
- *Trabalho e Energia*
- *Quantidade de Movimento*

## ☕︎ Objetivo

Os códigos servem como material de estudo e prática para reforçar os conceitos abordados na disciplina, incluindo simulações interativas como a Calculadora de Projétil e a Calculadora de Cargas Elétricas.

## ☕︎ Tecnologias Utilizadas

- HTML5;
- CSS3;
- JavaScript.

## ☕︎ Como Usar

## Como clonar apenas esta pasta

Este repositório contém todas as disciplinas do 2° ano. Se quiser baixar **somente** a pasta "Física", use o sparse-checkout do Git:

```bash
git clone --no-checkout https://github.com/MariaCampanaP/Segundo-Ano-de-CCOMP.git
cd Segundo-Ano-de-CCOMP
git sparse-checkout init --cone
git sparse-checkout set "Física"
git checkout main
```
